# Neovim v0.9.0

## Installation

### Requirements

_I'm using [asdf-vm](https://asdf-vm.com/) to install dependencies._

- Neovim (version: 0.9.0)
- Lua (version: 5.4.4)
- Ripgrep (version: 13.0.0)
- Fzf (version: 0.30.0)
- Githubcli (version: 2.14.1)

### Installing plugins

Go to

> cd ~/.config/nvim/

Open **nvim** and run the command to install all plugins.

>
> Shift + i
